# Energy Control Center

Односторінковий освітній застосунок для інтерактивного уроку з фізики. Працює у двох режимах: **Мультиборд** (великий екран) та **Інструмент Експерта** (смартфони).

## Швидкий старт

1. Встановіть залежності:
   - `npm install`
2. Запуск у режимі розробки:
   - `npm run dev`
3. Збірка для продакшну:
   - `npm run build`

## Технології

- React + Vite
- Tailwind CSS
- zustand (persist)
- dnd-kit
- recharts
- framer-motion

## Режими

- **МУЛЬТИБОРД**: балансування енергосистеми, лабораторія екранування, керування реактором.
- **СМАРТФОН**: калькулятор екранування для учнів.

## Розгортання

Автоматичний деплой на GitHub Pages через GitHub Actions.
